# Forus · 属于我们的时间 💙

记录我们的时间里程碑、照片，以及之后的种种。

> 在野在野！

## 技术栈

- **Nuxt 3** + **Vue 3** + **TypeScript**
- **Tailwind CSS**（晴空·薄荷配色：蓝 / 白 / 青）
- **@vueuse/motion**（滚动动效）

## 第一版功能

- 🏠 **首页**：在一起天数 + 纪念日倒计时 + 最近回忆 + 照片预览
- 📅 **时间线** `/timeline`：所有重要里程碑
- 🖼️ **照片墙** `/photos`：瀑布流照片展示

## 本地运行

```bash
# 安装依赖
npm install

# 启动开发服务器（默认 http://localhost:3000）
npm run dev
```

## 如何修改内容

所有内容都集中在 `data/` 目录，改这里即可，无需动页面代码：

| 文件 | 内容 |
|------|------|
| `data/site.ts` | 网站名、你们的名字、恋爱起始日、首页文案、下一个纪念日 |
| `data/milestones.ts` | 时间线里程碑（增删改节点） |
| `data/photos.ts` | 照片（把 `url` 换成你们的真实照片地址） |

### 配色

配色定义在 `tailwind.config.ts` 的 `colors` 中：

- `sky` 主蓝 `#2563EB`
- `cyan` 青 `#06B6D4`
- `mist` 浅青背景 `#ECFEFF`
- `ink` 墨蓝文字 `#0F172A`

## 目录结构

```

├─ pages/          页面（首页 / 时间线 / 照片墙）

├─ layouts/        布局

├─ utils/          工具（日期计算）
├─ types/          类型定义
├─ assets/css/     全局样式
└─ public/         静态资源
```
